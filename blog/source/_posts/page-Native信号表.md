# 常见信号

|   signal  |   code    |   describe    |
|   ---     |   ---     |   ---    |
|   SIGILL  |   4   |   非法指令    |
|   SIGTRAP |   5   |   断点调试异常 | 
|   SIGABRT |   6   |   异常退出    |
|   SIGBUS  |   7   |   总线错误    |
|   SIGFPE  |   8   |   浮点错误    |
|   SIGEGV  |   11  |   无效引用    |
|   SIGPIPE |   13  |   管道错误    |
|   SIGTERM |   15  |   终止运行    |
|   SIGSTKFLT|  16  |   栈异常      |
|   SIGSYS  |   31  |   系统调用异常 |
